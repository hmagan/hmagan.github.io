About

This website is for my IB MYP Personal Project, focusing on scientific and technical innovation. It is designed to test the user's memory and cognitive abilities through a variety of tasks and tests. This website is completely made from scratch using jQuery/JavaScript in addition to HTML/CSS. It should be noted that if you are not using Chrome, there is a decent chance this site will not function properly. Below is a list of each game:

  1. The "Chimp" Test: A test designed to evaluate a user's sequential memory. The name is inspired from a study conducted in which the test was performed by chimpanzees. They proved to excel at the test—can you beat them?
  2. The "n-back" Test: Designed after this study. Tests a user's working memory by assessing whether they can identify if the current stimuli and the stimuli n tests back.
  3. The "Simon" Test: Modeled after the Simon game from Hasbro. It takes a simple and straightforward approach to evaluate memory and how people react to stimuli. The user just needs to repeat the sequence that is displayed to them.
  4. Numbers Memory Test: Another simple test meant to evaluate memory. This test has a timer, and even though the numbers grow in size, the time given remains constant. This adds another level of difficulty.
